Change Log - Version 1.0
วันที่: 17 กุมภาพันธ์ 2026 Added ฟีเจอร์ใหม่
1. ระบบลบบัญชีผู้ใช้ (Account Deletion)
- เพิ่มระบบการลบบัญชีตนเอง (Self-Deletion) ผ่านหน้าโปรไฟล์
- ผู้ใช้งานสามารถเลือก "รับสำเนาข้อมูล (CSV)" ก่อนการยืนยันลบ หรือเลือกไม่รับก็ได้
-เมื่อยืนยันการลบแล้ว บัญชีจะถูกลบถาวร ไม่สามารถ Login ได้อีก และต้องสมัครใหม่เท่านั้น
2. ระบบนำทางและ Checkpoint สำหรับคนขับ (Driver Navigation)
- เพิ่มปุ่ม "เริ่มต้นการเดินทาง" และ "สิ้นสุดการเดินทาง" เพื่อควบคุมสถานะของการเดินทาง
- ระบบตรวจสอบตำแหน่งปัจจุบันผ่าน GPS เพื่อยืนยันการถึงจุดเริ่มต้น จุดแวะพัก และจุดสิ้นสุด โดยจะต้องอยู่ในรัศมีไม่เกิน 500 จากจุดนั้น
- เมื่อ Checkpoint สำเร็จ ระบบจะอัปเดตแผนที่เส้นทางจากจุดปัจจุบันไปยังจุด Checkpoint ถัดไปโดยอัตโนมัติ
- พิ่มระบบแนะนำเส้นทาง (Multiple Routes) ให้คนขับสามารถเลือกเส้นทางที่เหมาะสมที่สุดก่อนเดินทาง
3. ระบบบันทึกประวัติการใช้งาน (User Activity Logs)
- บันทึก Log ทุกกิจกรรมสำคัญของผู้ใช้งานรายบุคคลตามกฎหมาย พรบ.คอมพิวเตอร์
- เพิ่มหน้า Dashboard สำหรับ Admin เพื่อตรวจสอบ Log ย้อนหลัง
- รองรับการ Export ข้อมูล Log ออกมาเป็นไฟล์ เพื่อให้ผู้ใช้นำไปใช้เป็นหลักฐานทางกฎหมาย (Legal Purpose)


Change Log - Version 1.1
วันที่: 3 มีนาคม 2026 Added ฟีเจอร์ใหม่
1. ระบบลบบัญชีผู้ใช้ (Account Deletion)
- เพิ่มระบบการลบบัญชีตนเอง (Self-Deletion) ผ่านหน้าโปรไฟล์
- แก้ไขส่วนการดาวน์โหลดข้อมูล เป็นการเลือกได้ว่าจะลบรับข้อมูลอะไรบ้าง เช่น ประวัติส่วนตัว ประวัติการสร้างเส้นทาง ประวัติการเดินทาง และประวัติการสร้างยานพาหนะ(กรณีเป็นผู้ขับ)
- แก้ไขส่วนการรับไฟล์ข้อมูลจากการดาวน์โหลดที่หน้าเว็บไซต์เป็นการส่งไปยังอีเมลบัญชีของผู้ใช้
-เมื่อยืนยันการลบแล้ว บัญชีจะถูกลบถาวร ไม่สามารถ Login ได้อีก และต้องสมัครใหม่เท่านั้น
-เพิ่มการเช็คถ้าหากมีการเดินทาง หรือการสร้างเส้นทางติดค้างอยู่จะไม่สามารถลบบัญชีได้

2. ระบบนำทางและ Checkpoint สำหรับคนขับ (Driver Navigation)
- แก้ไขในส่วนการจองการเดินทางของผู้โดยสาร ให้ไม่สามารถจองการเดินทางได้หากจุดที่ต้องการขึ้น-ลงนั้นไม่อยู่ในรัศมี 500 ของจุดแวะของคนขับ
- เพิ่มปุ่ม การยืนยันการเดินทาง การยกเลิกการเดินทาง การปฏิเสธการยืนยันการเดินทาง และการปฏิเสธการยกเลิกการเดินทาง สำหรับผู้โดยสาร
- เพิ่มปุ่ม รับผู้โดยสาร และยกเลิกการเดินทาง สำหรับคนขับเพื่อส่งไปยังผู้โดยสารรายคน
- เพิ่มการอัพเดทเส้นทางหากคนขับยืนยันการจองเส้นทางของผู้โดยสาร หากจุดนั้นอยู่ในรัศมี500เมตรของจุดแวะของคนขับ แต่ไม่ใช่จุดแวะของคนขับ เพื่อเสนอเส้นทางให้คนขับไปรับหรือส่งผู้โดยสารจากจุดนั้นได้
- แก้ไขการ Checkpoint เมื่อคนขับเริ่มต้นการเดินทางแล้ว จากนั้นกดCheckpoint จุดที่มีผู้โดยสารขึ้นแต่ยังไม่กดรับผู้โดยสาร จะไม่สามารถ Checkpoint ที่จุดนั้นได้ จะต้องรับผู้โดยสารเสร็จสิ้นก่อน
- เพิ่มสถานะการเดินทางของผู้โดยสาร
- เพิ่มฟังก์ชันการกดรับผู้โดยสาร และการยกเลิกการเดินทาง โดยมีการโต้ตอบกันระหว่างคนขับและผู้โดยสาร รวมทั้งมีการแจ้งเตือน
- เพิ่มปุ่มสิ้นสุดการเดินทางให้ผู้โดยสาร จะกดได้ก็ต่อเมื่อคนขับได้checkpoint จุดที่ผู้โดยสารต้องการลงเรียบร้อย

3. ระบบบันทึกประวัติการใช้งาน (User Activity Logs)
- เพิ่มการตรวจสอบการแก้ไขไฟล์ Log
- เพิ่มการเลือกข้อมูลก่อนที่จะ Export แบ่งออกเป็น
  1.ข้อมูลส่วนบุคคล
      1.1ประวัติส่วนตัว
      1.2ประวัติการเดินทาง
      1.3ประวัติการสร้างเส้นทางและข้อมูลรถยนต์ (กรณีเป็นผู้ขับขี่)
  2.วันที่ เริ่มต้น จนสิ้นสุด
  3.ประเภทการกระทำของผู้ใช้งาน ตามกฎหมาย PDPA และ พรบ.คอมพิวเตอร์ (เพิ่มการจัดเก็บ Type ในฐานข้อมูล)
    1. Authentication & Access Logs (Log การเข้าถึง)
    2. Transactional & Activity Logs  (Log การทำรายการ) 
    3.Navigation & Behavioral Logs (Log พฤติกรรม)
    4.. Security & Audit Logs (Log การตรวจสอบความปลอดภัย)
